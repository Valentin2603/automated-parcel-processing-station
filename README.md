# Automated Parcel Processing Station

Учебный Java-проект, моделирующий работу автоматизированной станции обработки посылок с использованием многопоточности.

## Описание

Программа имитирует систему, где поток-генератор создаёт посылки, а несколько рабочих устройств параллельно обрабатывают их по направлениям.

Каждая посылка имеет:

* номер;
* направление обработки;
* приоритет.

Устройства забирают из общего хранилища только посылки своего направления.
Сначала обрабатываются посылки с более высоким приоритетом, а при одинаковом приоритете сохраняется порядок поступления.

## Возможности

* генерация посылок в отдельном потоке;
* ограниченное общее хранилище;
* несколько направлений обработки;
* несколько устройств на каждое направление;
* приоритетная обработка посылок;
* мониторинг состояния хранилища и устройств;
* корректная остановка потоков после заданного времени работы.

## Используемые технологии

* Java
* `Thread`
* `volatile`
* `ReentrantLock`
* `Condition`
* `AtomicBoolean`

## Архитектура

Основные классы:

```text
Main              запуск программы, создание потоков и мониторинг
Storage           общее хранилище посылок
Device            рабочее устройство обработки
Package           модель посылки
StorageSnapshot   снимок состояния хранилища
DeviceSnapshot    снимок состояния устройства
```

## Что демонстрирует проект

Проект показывает практическое применение многопоточности в Java:

* создание и запуск потоков;
* синхронизация доступа к общему ресурсу;
* ожидание условий через `Condition`;
* паттерн producer-consumer;
* работа с приоритетами;
* безопасное получение snapshot-состояния;
* завершение потоков через `interrupt`.

## Запуск

Скомпилировать проект:

```bash
javac src/*.java
```

Запустить:

```bash
java -cp src Main
```

## Статус

Проект выполнен в учебных целях для закрепления многопоточности, синхронизации потоков и работы с общими ресурсами в Java.
